# 🚀 TaskFlow Pro

A modern interactive task manager built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks complete
- Filter tasks
- Search tasks
- Dark/light mode
- localStorage persistence
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download project
2. Open index.html
3. Start managing tasks
